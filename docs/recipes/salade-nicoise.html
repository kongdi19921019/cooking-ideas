Salade Niçoise

⏱ 30 min 👥 Serves 4
Watch Julia Child make it on YouTube

Steps

  1. Sear tuna 1-2 minutes each side. Rest and slice.
  2. Blanch green beans 3 minutes, cool in ice water.
  3. Arrange all components on a platter.
  4. Drizzle generously with vinaigrette just before serving.
Julia's tip: Dress each component separately before arranging — this way every element is properly seasoned.

Serve With

  • Crusty baguette
  • Rosé Provençal
